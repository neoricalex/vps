== Installation ==

1. Unpack plugin zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open plugin settings page
4. Create a Smartarget account or log in to yours
5. Add your website and configure Whatsapp - Contact Us app for your website
6. Go to Integration and copy the User ID to the input text box to this page
